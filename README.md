# BetterBlizzCC
BetterBlizzCC displays CC's on the Player in the center of the screen similar to how it works on Retail but with a few improvements like showing Interrupt duration while Full CC'ed and which school you got interrupted on. It can be moved, resized and put on "Icon Only" mode in the settings.

`/bbcc` to open settings.

Note: This addon is intended for TBC and Mists of Pandaria and PvP only. It works for Cataclysm too but there might be some missing CC's. Might update it in the future to support PvE CC as well.

This is a stand-alone version of the LossOfControl setting from my other addon BetterBlizzFrames.
